# HTML Presentation Deck Skill

This repository contains the `html-presentation-deck` Codex skill.

It helps Codex create, redesign, verify, and deploy web-based presentation decks such as React/Vite HTML slides, frontend slide decks, animated product-launch decks, GitHub Pages presentations, and browser-printable PDF decks.

## Install

Install the skill from this GitHub repository:

```bash
npx skills add https://github.com/water-nn/html-presentation-deck-skill --skill html-presentation-deck
```

Then confirm it is available:

```bash
npx skills list
```

## What It Supports

- React / Vite / TypeScript web slide decks
- 16:9 centered presentation stage
- Keyboard navigation
- Hover Agenda navigation
- Fullscreen mode
- Progress UI
- Dynamic background systems
- Chart and data-story components
- Theme variants such as `editorial` and `visual`
- Tone presets such as champagne and aurora
- Local double-click preview scripts
- GitHub Pages deployment workflow
- Browser print / PDF export guidance
